# Web Manajemen Seminar

Web manajemen seminar untuk kantor untuk melakukan training karyawan.

## Fitur:
- Perusahaan bisa menjadwalkan training dan karyawan bisa mendaftar.
- Admin mengelola daftar peserta, materi training, dan feedback.
- Ada autentikasi user, setiap karyawan bisa memiliki akun sendiri dan mendaftar dengan akunnya 
- CRUD training & peserta, laporan kehadiran.

## Teknologi yang Digunakan
- PHP dengan pdo untuk Backend
- MySQL untuk databasenya
- Bootsrap + SBadmin template untuk ui
